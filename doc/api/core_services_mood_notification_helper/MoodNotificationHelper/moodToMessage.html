String moodToMessage(
  1. String mood
)

Selects a supportive message corresponding to the given mood.

Returns a randomly chosen encouraging message from a predefined list associated with the specified mood.

If the mood is not recognized or no messages are defined, a default reminder message is returned instead.

The randomness helps prevent repetitive notifications.
