Getting Started

🔧 Installation & Setup

1️⃣ Clone Repository

git clone https://github.com/your-username/wellcare.git
cd wellcare

2️⃣ Install Dependencies

flutter pub get

▶️ Run Application

flutter run

🔄 หากมีการ git pull อัปเดตโปรเจกต์

หลังจาก pull โค้ดล่าสุดแล้ว ให้รัน:

flutter pub get
flutter run

หากพบปัญหา build error ให้ลอง:

flutter clean
flutter pub get
flutter run

⚠️ หมายเหตุ: โปรเจกต์นี้เชื่อมต่อกับ Firebase ของอานา
หากต้องการใช้งาน Firebase ของตนเอง กรุณาสร้างโปรเจกต์ใหม่ใน Firebase Console และเปลี่ยนไฟล์ google-services.json / GoogleService-Info.plist เอง
